We are now showing how to setup a Master/ Detail relationship using @Input in the child component to recieve a value from the parent component.

The HeroDetailComponent will present details of a selected hero.

[hero]="selectedHero" This sets the [hero] property of the DOM (this maybe an html or template property not the same as an html element) to the 'selectedHero' variable of this class.

We are passing the Master component value down to the child template "@Input hero: Hero"
